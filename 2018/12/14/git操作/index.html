git master让其他人开发你的项目

  • 在git lap里面选择一个项目
  • 点击左侧的settings
  • 点击左侧的members
  • 点击右侧的绿色按钮 add members
  • 在people里选择一个人
  • 在 product access 选择 developer(开发者的意思)
  • 在点击 add users to project

把主分支的项目拉下来
